Kiểu dữ liệu và toán tử trong JavaScript

January 14, 2025 · 1 min
Table of Contents

1. Kiểu dữ liệu trong JavaScript#

JavaScript hỗ trợ nhiều kiểu dữ liệu như:

  • Number
  • String
  • Boolean
  • Object
  • Array

2. Toán tử#

JavaScript có các toán tử:

  • Toán tử số học: +, -, *, /
  • Toán tử so sánh: ==, ===
  • Toán tử logic: &&, ||

Việc hiểu rõ kiểu dữ liệu giúp lập trình hiệu quả và tránh lỗi.
