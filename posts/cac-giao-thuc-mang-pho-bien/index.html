Các giao thức mạng phổ biến: HTTP, HTTPS, FTP, DNS

January 13, 2025 · 1 min
Table of Contents

  • 2. FTP

1. HTTP và HTTPS#

HTTP là giao thức truyền tải dữ liệu trên Web.
HTTPS là phiên bản bảo mật hơn, sử dụng mã hóa SSL/TLS.

2. FTP#

FTP dùng để truyền file giữa máy client và server.

3. DNS#

DNS giúp chuyển đổi tên miền (ví dụ google.com) thành địa chỉ IP để máy tính có thể giao tiếp với nhau.

4. Vai trò trong lập trình#

Các giao thức này là nền tảng cho Web, API và các hệ thống Client–Server.
