Xử lý Exception trong Java

January 4, 2025 · 1 min
Table of Contents

1. Exception là gì?#

Exception là các lỗi xảy ra trong quá trình thực thi chương trình.

2. Cách xử lý Exception#

Java sử dụng khối try - catch.

try {
    int a = 10 / 0;
} catch (Exception e) {
    System.out.println("Có lỗi xảy ra");
}

## 3. Ý nghĩa
Giúp chương trình không bị dừng đột ngột khi gặp lỗi.

## 4. Kết luận
Xử lý Exception giúp chương trình ổn định hơn.
